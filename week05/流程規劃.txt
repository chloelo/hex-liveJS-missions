流程規劃

- 產品列表頁
  - api 撈資料 --v
  - 產品列表 --v
  - 查看更多
    - 取得單一產品 api
    - 加到購物車
      - 依選取的數量加到購物車列表，更新購物車 api
      - 購物車頁顯示品項列表
  - 加到購物車
    - 新增 1 筆品項到購物車列表

- 購物車頁
  - 取得購物車 api
  - 購物車列表
  - 增加/減少品項數量
    - 取得選取購物車
    - 修改數量，更新購物車 api
    - 重新渲染購物車列表
  - 總計金額
    - 加總所有所有品項與相對應數量的金額，顯示於畫面
  - 刪除品項
    - 取得選取購物車
    - 刪除當筆購物車，更新購物車 api
    - 重新渲染購物車列表
  - 刪除所有品項
    - 撈購物車 api 刪除所有品項
    - 重新渲染購物車列表
  - 表單驗證
    - 使用驗證表單套件，格式正確就送出表單
    - 新增訂單，更新訂單 api
