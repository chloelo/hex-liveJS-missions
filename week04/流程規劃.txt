流程規劃

- 登入頁
  - 發送 API 驗證 -- v
  - 儲存 token 到 cookies -- v
  - 轉頁到產品列表  -- v

- 產品列表頁
  - api 撈資料 -- v
  - 產品列表 -- v
  - 新增產品
    - 新增產品，更新遠端 api 
    - 重新渲染產品列表
  - 編輯產品
    - 取得單一產品 api
    - 修改當筆資料，更新遠端 api
    - 重新渲染產品列表
  - 刪除產品
    - 取得選取的產品
    - 刪除當筆資料，更新遠端 api
    - 重新渲染產品列表
  - 分頁
    - 取得分頁的 api -- v
    - 了解分頁資料的結構去做切換頁 -- v
